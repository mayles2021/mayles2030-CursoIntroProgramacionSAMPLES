
- Comandos Útiles de MS-DOS

	- Borrar Terminal de comandos => cls
	- Borrar contenido de directorios => delete *.*

	- Entrar en carpeta => cd NombreCarpeta
	- Salir de carpeta => cd ..

	- Crear carpeta => md NombreCarpeta
	- Borrar carpeta => rd NombreCarpeta


- Ver Ayuda y Version dotnet instalada

	1. dotnet

	2. dotnet --version


- Crear y Modificiar código VSCode

	1. Abrir terminal 
	2. Crear proyecto

		dotnet new console -o AppName

	3. Entrar en el proyecto

		cd AppName

	4. Abrir proyecto

		code .


- Arrancar proyecto VSCode

	1. dotnet run


- Compilar proyecto VSCode

	1. dotnet build
